# Assets Directory

Place application icons here:

- **icon.png** - Application icon (512x512 recommended)
- **icon.ico** - Windows icon (optional, will be generated from icon.png)
- **icon.icns** - macOS icon (optional, will be generated from icon.png)

The application will work without custom icons, falling back to default Electron icon.

To add a custom icon:
1. Create a 512x512 PNG icon
2. Name it `icon.png`
3. Place it in this directory
4. Rebuild the application

Icon requirements:
- Format: PNG
- Size: 512x512 pixels (or larger, square aspect ratio)
- Transparent background recommended
- Minecraft-themed design recommended
