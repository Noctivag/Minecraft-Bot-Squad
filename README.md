## ğŸ’» Usage Examples

### Basic Usage

```javascript
const { MinecraftBotSquadPlugin } = require("./plugin");

const plugin = new MinecraftBotSquadPlugin({
  host: "localhost",
  port: 25565,
  botCount: 5
});

await plugin.initialize();
// Bots now work autonomously!
```

### Build Structures

```javascript
// Build a cottage
await plugin.buildStructure("Blaze", "cottage",
  { x: 100, y: 64, z: 100 });

// Build an enchanting room
await plugin.buildStructure("Blaze", "enchanting_room",
  { x: 120, y: 64, z: 100 });
```

### Create Redstone

```javascript
// Build item sorter
await plugin.buildRedstone("Fiona", "item_sorter",
  { x: 150, y: 64, z: 100 });

// Build mob grinder
await plugin.buildRedstone("Fiona", "mob_grinder",
  { x: 200, y: 64, z: 200 });
```

### Monitor Progress

```javascript
// Show all bots progress
plugin.showProgress();

// Show specific bot
plugin.showProgress("Alex");

// Get detailed stats
const status = plugin.getStatus();
```
